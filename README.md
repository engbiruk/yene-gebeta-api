# Yene-Gebeta API Project
> Restaurant Searching, Rating and Reviewing Platform 

# Introduction
> Yene-Gebeta is a restaurant rating and review application for Ethiopians. Yene-Gebeta brings local restaurants to a system that can make them available for any users who are looking for a place to eat. It keeps track of restaurant service providers and register them on a mobile app platform to yield a better discoverability.

# Objective
> The general objective of building this platform is to provide a service that can allows peoples to search for restaurant and get their location, review and rate.
